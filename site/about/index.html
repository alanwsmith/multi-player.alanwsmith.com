About This Project

This project is a recreation of a page I saw years ago. It used the same video from the band Pogo with the same filmstrip style movement. I can't find the original. So, I decided to recreate it.

Web Components

The page is built from two vanilla javascript web components. Using components is a bit overkill. I'm still new to them though. I'm a big fan getting reps under your belt on easier things so you've got better footing when they get harder. This project turned out to be perfect for that.

Syncing The Videos

The page automatically determines how may players to use based on the browser's window size. The videos start with a little delay between them. It works well most of the time on good network connections. Sometimes they don't start when they should and things look a little wonky. If you're on a solid network connection refreshing the browser might make things smoother.

Two Patterns

If you look at the source code you'll see a commented out section for another pattern. It's more of star where things expand out from a central frame. I liked the idea, but it didn't work as well in practice.

I went back to the filmstrip style. I might play around with other patterns at some point. I like the way it looks for now.
